<h1 align="center">SIRATA (Sistem Informasi Rukun Tetangga)</h1>

<p align="center">Aplikasi berbasis web untuk memudahkan pekerjaan RT dalam mengolah data kependudukan.</p>

## Features

- [ ] Pengguna
- [ ] Informasi Penduduk
- [ ] Tambah Penduduk
- [ ] Penduduk Pindah
- [ ] Penduduk Meninggal
- [ ] Filter Penduduk berdasarkan:
  - [ ] Jenis Kelamin
  - [ ] Jenis Pekerjaan
  - [ ] Pendidikan
  - [ ] Usia
